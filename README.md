# Runt

Este repositorio contiene el proyecto `CE0864_Consulta_Runt_Simit`, una automatización para consultas en el portal SIMIT.

## Documentación agregada

- `CE0864_Consulta_Runt_Simit/README_INSTALACION_CMD.md`
  - Manual de instalación y ejecución usando Windows CMD.
- `CE0864_Consulta_Runt_Simit/INSTRUCCIONES_CAPTURA_HTML.md`
  - Instrucciones para capturar HTML y verificar selectores del portal.

## Carpeta principal del proyecto

- `CE0864_Consulta_Runt_Simit/Aut. Simit x Placa y Cedula/`
  - `main.py`
  - `capturar_html_simit.py`
  - `Funciones/f_navegar_simit.py`

## Instrucciones rápidas

1. Abre `CE0864_Consulta_Runt_Simit/README_INSTALACION_CMD.md`.
2. Sigue los pasos para instalar dependencias y ejecutar la automatización.
3. Si necesitas calibrar selectores, usa `capturar_html_simit.py` y `INSTRUCCIONES_CAPTURA_HTML.md`.
